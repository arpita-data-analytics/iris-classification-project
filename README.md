# Iris Flower Classification 🌸

This project uses a **Machine Learning model** to classify Iris flowers into three species based on their measurements.

## Dataset

The Iris dataset contains 150 samples with the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Target classes:

* Setosa
* Versicolor
* Virginica

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Machine Learning Model

The model used in this project is **K-Nearest Neighbors (KNN)** for classification.

## Steps Performed

1. Import libraries
2. Load the dataset
3. Explore the data
4. Split data into training and testing sets
5. Train the KNN model
6. Make predictions
7. Evaluate model accuracy

## Project Files

* `iris_model.ipynb` → Jupyter notebook with the model
* `iris.csv` → Iris dataset

## Result

The model successfully classifies Iris flowers based on sepal and petal measurements.
